周末，清晨格外安静，朦胧的天正下着小雨，已踏上另一条颇具意义的旅程。

![又遇 Not Found 代号](./screennails/20230729_081330.webp)

本次目的地是红色根据地——井冈山。

位于吉安市的井冈山，地处赣湘两省交界的罗霄山脉中段，最高峰江西坳海拔1841米。上世纪二十年代末，毛泽东、朱德等老一辈无产阶级革命家，率领中国工农红军来到这里，开展了艰苦卓绝的井冈山斗争，创建了中国第一个农村革命根据地，“星星之火，可以燎原”。

![前往井冈山](./screennails/20230730_153217.webp)

![倒影中的金光灿灿五角星](./screennails/20230729_143244.webp)

茨坪是一座美丽的公园式山城，天街、挹翠湖、革命博物馆等构成了一道道富有特色的靓丽风景线。

![天街中的一间瓷器店](./screennails/20230729_211506.webp)

![天街中央的雕塑](./screennails/20230729_142237.webp)

天街集购物、餐饮与红色经典于一体，这里的房子大多为白墙青瓦，新遂边陲特别区工农兵政府旧址的墙体则为黄色，可免费参观，外面步行街上许多金色的大型五角星构成的走廊可从中穿越，终点是一个万丈光芒的红色五角星。由于刚下过雨，色彩仿佛都经过重新涂抹，一切都是那么的清新自然，犹如游历书画当中。

![挹翠湖一景](./screennails/20230730_104008.webp)

![挹翠湖及远处火炬](./screennails/20230729_204722.webp)

挹翠湖风景秀丽，坐落山中，四周绿荫覆盖，湖面如镜，其内有小岛，亭台楼阁错落有致点缀其中，石桥步道相连，在周围景致包围下尤显精致，夜晚更是五光十色，即便盛夏也是凉爽宜人，人来人往，或是拍照打卡、或是闲来相聚、或是途中经过，欢歌笑舞、络绎不绝。

![井冈山革命博物馆重点一处场景](./screennails/20230729_155422.webp)

![井冈山革命博物馆展物](./screennails/20230729_161149.webp)

井冈山革命博物馆气势恢宏，里面从南昌起义开始，到走进井冈山、会师、反会剿等，耳熟能详的红色历史篇章生动地呈现眼前，地动山摇，忆苦思甜，井冈山斗争让我们对历史进行了深刻思考，让我们知道现在的幸福生活来之不易。

![毛泽东同志旧居之外景](./screennails/20230730_102922.webp)

![毛泽东同志旧居之卧室](./screennails/20230730_095803.webp)

毛泽东同志旧居里面有当年红军第四军军部旧址等众多重要历史遗迹，里面更是包含了许多思想开拓和精神指引。艰苦的环境，却诞生了伟大的奇迹。

![特色冰棒](./screennails/20230729_165032.webp)

当然，门口也少不了网红冰淇淋（冰棒），设计独特很应景，价格虽然不便宜，但还是忍不住每款都买了尝尝。

![谢氏慎公祠](./screennails/20230730_145435.webp)

![卧室](./screennails/20230730_143919.webp)

位于茅坪有一栋土砖结构的楼房，二层顶部有个方便采光的八角形天窗，因而得名八角楼。在这里，涌现了许多反应当时客观事实的理论和实践，更提出了“工农武装割据”的光辉思想。

这次井冈山之旅，加强了对历史深刻教训的回忆，以及对未来美好憧憬的向往，时刻提醒自己，牢记使命，不忘初心。
