## 厦门

厦门（Amoy）也是个常来的地方，城区面积不大，近海岛屿，风景特色独到。其位于濠门（海门东北）以南，南为下，门乃江河入海有山崖处之地名通称，经文人雅化而得现名。

![墙上满是花](./screennails/20240214_101432.webp)

华新路上，近代建筑比比皆是，小路两旁是红砖石墙，鲜花四溢，时不时一群游客驻足拍照留念，却很宁静。

![新晋网红打卡点](./screennails/20240214_103447.webp)

远离喧嚣的小街巷，这里曾有取景拍摄过多部电影作品，足显文艺范，其给人的诗意感觉的确也与众不同。

![路边小店](./screennails/20240214_104716.webp)

饰品店与咖啡馆在其中零星点缀，门面也都经过精心布置，各有不同，虽此刻人不多，但感觉一切并不怠慢。恰逢情人节期间，一些饮品店开始有情侣八折优惠。

![可爱的龙](./screennails/20240214_105103.webp)

同样，龙年氛围也随处可见。继续游走街头，不算是 city walk，全然当作是散心。旁边有个花鸟市场，还有中山公园和歌剧院。

![中山公园一景](./screennails/20240214_111545.webp)

公园中，绿地在小湖水间散步，庭院遍布，并设有一博物馆，小桥流水绿植花香，初春如入夏，绿意盎然。

![沙茶面](./screennails/20240214_120425.webp)

中午，思明东路一必吃榜餐厅来一顿当地特色海鲜美食，各款菜肴色香味美，看着窗外惬意的老城风景也是很有风味。

![古城西路](./screennails/20240214_114642.webp)

老城区道路较狭窄，人车分流亦不明显，开车需多加留意，路面时有坡度，这种特别干净版的市井氛围让人感到非常有港片质感。

![思明南路路边建筑](./screennails/20240214_130646.webp)

靠近步行街附近，人流量明显很大，周围的建筑颇具南洋风格，商铺餐饮遍布，感觉多年来变化并不大。

![靠近沙滩](./screennails/20240214_132443.webp)

下午时分沿岛开行半圈，望海观岛。路上较为拥堵，但也好，正好可以在车上看看海景。至白城沙滩附近，驻车玩赏。

![海水扑向沙滩](./screennails/20240214_134122.webp)

走上沙滩，径直迈向海边，蓝色大海不停用浪花潮水冲刷着岸边，沙硕和水面都反射出晶莹剔透的亮斑光点，远处的船只若隐若现，有邮轮、有军舰。

![海中浪花](./screennails/20240214_141414.webp)

淡淡蓝灰色的天空中，透出远处岛屿山脉，下面是湖蓝色海水，靠近是层层白色浪花和绿色水浪，近处是黄色的湿沙地，后面是金黄色的细沙地。

![白城沙滩](./screennails/20240214_142811.webp)

阳光照射和海水反射下，颜色分明且非常艳丽。沙滩上人很多，嬉戏打闹。可能因为还没到夏天，下水游泳的人并不多。

![厦门大学校门](./screennails/20240214_143115.webp)

时间不早了，离开沙滩，对面便是厦门大学，此前进去参观过多次，这次就不再打算进去了，还记得里面文创点的谐音梗“我是厦（吓）大的”。

![厦门大学一角](./screennails/20240214_143044.webp)

今天据说也是迎财神日，多个节日碰撞到一起，出来玩的人自然也多，因此还是以简洁休闲为主。

![特色奶茶](./screennails/20240213_220923.webp)

当然，当地特色奶茶也还是要尝一尝的。今天又是开心快乐的一天！

## 福州

![市区](./screennails/20240215_090208.webp)

福州（Hockchew）别称榕城，因境内一座福山而得名。今日再次来访旅游，但时间匆忙，仅有一天，只能住在市区，仅去少数几处观光。

![军舰](./screennails/20240215_104242.webp)

马尾素有“千年古港、百年船政”之称，一早即开往，路上滨江大道绿化较好，抵达后停好车，一艘老旧军舰在此已停放许久。

![大炮模型](./screennails/20240215_105003.webp)

一江两岸，对面小山和建筑只可见其蓝色剪影，出海江面上是它们的同色投影，像淡雅水墨画。这边几尊模型大炮，旁边是一些花坛帐篷，船政博物馆也在附近。

![罗星塔和村七娘](./screennails/20240215_112039.webp)

沿着江边步道上爬，很快就到山顶，罗星塔矗立于此，始建于南宋。周围是个小广场，许多近代史名人雕塑。

![罗星塔公园一景](./screennails/20240215_114149.webp)

公园中还散落有许多有趣的装饰，在一派严肃中增添了许多轻松诙谐的色彩，更还可供游客拍照打卡。

![三坊七巷](./screennails/20240214_185324.webp)

下午抵达位于鼓楼南后街的三坊七巷，此处发源于晋代，规模较大，人潮涌动非常繁华，各类店铺密织其间。

![小巷中](./screennails/20240214_192751.webp)

即便走进其中深巷，也是别有洞天，白墙青瓦木门窄道灯笼高挂，却藏匿不住大树参天；大量灯光运用及少许现代几何图形嵌入其中，也是让你确信自己并未穿越。

![瓷器店](./screennails/20240214_192650.webp)

随意走进其中一间大宅，装饰精致古朴，颇为讲究。

![青墙](./screennails/20240214_190721.webp)

高枝树叶下青砖高墙，是吉庇巷上建筑特色，顶部精致红色木制顶檐显示出大气端庄设计中保留着细腻。

![餐厅](./screennails/20240214_205204.webp)

是时候吃个晚饭了，走进其中一家金碧辉煌的本地餐厅，品尝闽菜美食，也是旅行一大重要事项，不可或缺。吃在中国，也是一种文化。

![佛跳墙](./screennails/20240214_201218.webp)

海鲜必不可少，一道道福建特色菜肴，味道鲜美。想到前一段时间疫情居家隔离，家中翻出的一箱福建名菜佛跳墙成了当时缺粮少菜时的救命稻草，回想起来也是一番趣事，这次在福州自当专门特地来品尝啦。

![当地特色奶茶](./screennails/20240214_214223.webp)

当然，依然还是要品饮当地特色奶茶的，作为夜宵饮料。
