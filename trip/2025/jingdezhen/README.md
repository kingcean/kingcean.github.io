之前数次经过景德镇，但都没有好好坐下，看着骨瓷玲珑，品着清茶油菜花，感受这古老艺术带来的清新体验，这次终有机会下定决心到此一游。

![佑陶灵祠](./screennails/20250705_143112.webp)

在这停留时间并不长，只有短短数日，安排很轻松，这座城市坐落于低丘山林之间，但仍然抵不住炎炎烈日蒸拿，每天只能下午避开垂直直射后出行游览。

## 三宝村

![三宝蓬美术馆](./screennails/20250706_142550.webp)

住在陶源谷，这附近到处都是陶瓷艺术观览制作。对面有个美术馆，低矮几何型建筑物内，成列着一些精美艺术品，包括许多瓷画。

![流水](./screennails/20250706_155700.webp)

美术馆所在艺术聚落内，有小草坡和小河流，修饰休闲精致，可以慢慢步行参观，内有许多体验馆。

![茶杯](./screennails/20250706_143401.webp)

还有许多喝茶休憩的地方，在设计上也很贴靠瓷器风格，非常别致。

![茶馆](./screennails/20250706_155135.webp)

来到一个茶馆，坐下来休息，古色古香，中式装饰，到处都有精致瓷器展示。

![陈皮普洱](./screennails/20250706_145857.webp)

在床边落座，看着外面的绿色和流水，点了几壶不同的茶，来电甜点小吃，慢生活中在空调房内也躲过了炎日。

![三宝路](./screennails/20250706_121728.webp)

休息得差不多了，走到三宝路上，整个景区基本都沿着这条路一字展开，可以乘坐观光车沿途参观停留，陶陶罐罐到处都是。

![胚](./screennails/20250705_111918.webp)

下午茶和陶艺都是酒店送的，坐下来进行体验。

![笔和颜料](./screennails/20250705_111550.webp)

在轻黄白色的陶罐中许多鲜艳多彩的颜色中雕琢刻画，慢慢制作自己的艺术品。

## 御窑厂

![遗址](./screennails/20250705_142802.webp)

次日下午，来到御窑厂参观，这里是古时候给宫廷制作官瓷的地方。

![龙珠阁](./screennails/20250705_143843.webp)

大草坪上有一座宝塔，四处都是绿油油的植物，其中也有许多遗址散步草坪上。

![御窑博物馆外景](./screennails/20250705_144339.webp)

博物馆是巨型红石拱形建筑物，远处有一高竖烟囱，非常有特色，古典在现代中重新演绎。

![御窑博物馆内](./screennails/20250705_152854.webp)

馆内陈列有许多精美瓷器，其中许多说破碎后重新粘合的，因为古时候御制贡瓷存有瑕疵的会直接销毁，现在恰好成为了展品，从某种程度呈现了彼时的工艺。

![瓷器风格制品](./screennails/20250705_144632.webp)

公园内还有许多周边商品，包括一些长得像瓷器的包或其它制品。

![窑山邑](./screennails/20250705_160405.webp)

博物馆后则是古街，里面也有许多瓷器相关的参观和购买。

![巷子](./screennails/20250705_160445.webp)

石砖屋间细长石砖路藏不住两遍郁郁葱葱的生机，人来人往大家拿着单反自拍杆不亦乐乎。

![徐家柴窑旁](./screennails/20250705_165131.webp)

高大烟囱旁的咖啡店和甜品铺，瓦木古屋尽显古朴风采。

## 陶溪川

![夜幕降临陶溪川](./screennails/20250705_203551.webp)

夜晚的陶溪川是新时代的陶瓷文化创意园，是现代文明中对景点的传承和敬意。

![瓷壶](./screennails/20250705_204338.webp)

许多当代精品在此展示出售，晶莹剔透纹路精美，体现出瓷器文化蓬勃发展的美好现状。

![烟囱](./screennails/20250705_204824.webp)

灯火辉煌与红墙石砖的搭配带来的是一种非常有艺术感的氛围，也是人海集中地的创造者。

![水池映象](./screennails/20250705_205307.webp)

水影中，魅力瓷都似若隐若现，又安安静静摆在那，让你深刻体会，这博大的艺术文化情怀。
