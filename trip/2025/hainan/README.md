# 海口

> 越过琼州海峡，海滩与火山，历史海滨城市海口，自带自然旖旎风光。

![海口](./screennails/20250510_172205_169.webp)

居然以前并没来过海南，原本计划这次节假日出游的，奈何游客肯定很多，便错峰出行，待小长假国内迁徙人群返程一周后，跨越一个周末游览数天。由于天数并不多，出于休闲奔波适中的基准定位，因此本次出游仅集中在海口及附近地区，而三亚等其余城市以后再做计划。

![海景房景观](./screennails/20250510_092849.webp)

乘飞机抵达美兰机场，从道路两旁椰子树中穿梭直奔酒店入住休息，赶在行政酒廊结束前匆匆再补充一些吃喝，第二天一早在海景公寓套房中醒来，早餐后便开始了一天的行程。

## 沙滩

![阳光沙滩](./screennails/20250512_155437.webp)

阳光明媚风和日丽，沙滩始终给人一种不可抗拒的力量，看似宁静，实则波澜，亦静亦动，伴随着潮汐微风声响。

![城市与海](./screennails/20250512_150747.webp)

汩汩浪花袭来，远处是大海，也有城市，黄色的沙滩，蓝色的海天，日复一日。

![海水冲滩](./screennails/20250512_153127.webp)

海水中带着大量泡沫，抚平沙滩上的坑洼，也抚平我们的内心，让一切平静。

![休闲沙滩](./screennails/20250512_172030.webp)

在沙滩边小憩，忘却时间流逝。

## 火山

![火山脚下](./screennails/20250510_120423.webp)

雷琼世界地质公园在城市的西南部，里面树木颇多，路面有许多冷却熔岩形成的石头组成和点缀，其间还有水池和瀑布，随着越往里走，地势开始上升，但总的来说还是非常平缓的，经过一组凉亭，眼前便是一片草坪，加上若干竖石与一尊金色“大鸡大利”雕像，这便是风炉岭火山脚下。

![即将登顶](./screennails/20250510_124025.webp)

沿着石阶一路向上爬，很快便到了火山口，风吹凉爽。沿着火山口边缘可以绕行一圈，但此起彼伏并不平坦，继续攀几十步登就是海口制高点了，海拔 222.8m，和基本以 km 为单位的各峰各岳相比这可以说是一座非常矮的山了，但很有特色。

![火山内部](./screennails/20250510_124112.webp)

站在制高点向外可以看到周围相当大范围的景观，而向内则可看到火山口全貌，这属于玛珥火山类型，总的来说火山口是个圆形。由于长期处于休眠状态，现如今可谓植被茂密，火山口内的栈道也是清晰可见。

![火山口内](./screennails/20250510_123016.webp)

沿着栈道可以抵达火山口内，站在巨型碗口中，有一个人造木制地板平台可以从内观赏，并可明显感到温度的下降。平台上立有一标识，说明洞深69m、大气层内距约 2000km。仰望洞口，可以看到周围一片盎然绿色。

## 沐心

![木屋](./screennails/20250510_155129.webp)

沐心石屋是一个火山石屋院落群，抵达前的道路两边都是由火山石堆筑的暗灰色围墙，一进门便是一桶桶酒罐，里面有各种当地酿造的酒。

![树丛中白色座椅](./screennails/20250510_160601.webp)

穿过大堂，是一个烧烤区，旁边一大片荔枝树林，内有小石路可供穿行游览，可以看到偶然间出现的休闲座椅，也能看到公鸡母鸡。

![石墙壁上的绿植](./screennails/20250510_160703.webp)

小石路通往各民俗小屋，均有火山石和木条制作，周围点缀许多绿植，别有特色、非常幽静。本次前来不是为了住在这，只是参观。

## 骑楼

![白色骑楼](./screennails/20250510_171810.webp)

许多城市都有骑楼老街，海口的也非常有名，一栋栋南洋风格的石制建筑沿街而建，蓝天白云下，繁茂的商业和餐饮店铺比比皆是。

![石拱](./screennails/20250510_175035.webp)

墙壁干净、雕刻精细，可以看到复杂的设计结构和装饰版面，但雨水对漆面冲刷留下的历史印记也是非常明显，见证着从以往到现在的繁华。路面上方挂着许多花灯，也有凤凰和蝴蝶等造型，增加了许多鲜艳的活力。

![店内茶壶](./screennails/20250510_173711.webp)

走进其中一家店，香溢花茶和精致杯壶在木制展柜上摆设得玲珑满目。

![夜间雨后的骑楼老街](./screennails/20250510_192039.webp)

而雨后的夜色下，骑楼老街更是显现出非常繁华市井的一面，灯笼和灯光照亮整块街区，靛色天空下，深色路面的湿润投影也呈现蓝黄冷暖的颜色，一副五彩缤纷的油画跃然眼前，刹那间仿佛来到了民国时期。

## 旅行继续

![乘环岛高铁前往海花岛](./screennails/20250511_105838.webp)

# 海花岛

> 海南西线明珠风光秀丽特色分明，各类风格建筑物和主题区显现出别样景观。

从海口乘坐环岛高铁一个小时后抵达白马井站下车，乘班车可以很快就到达海花岛。海花岛位于儋州市滨海新区，是个南海北部湾中的人工岛屿群，从天空往下看是一朵平铺于海洋中的美丽花朵。

![通往岛上的大桥](./screennails/20250511_130956.webp)

## 1号岛

海花岛主要由3座岛屿组成，中间的是1号岛，与周围两座有桥梁相连并由免费换乘巴士可接送。主岛（1号岛）正南方有一座桥与海南本岛相连，从海南本岛往海花岛方向看去，对称的两座孪生高楼和中间圆柱形带冠建筑，非常壮观。孪生高楼是酒店，本次旅行即住于此，从客房向外看去，两条常思带中间一条柏油马路直通海峡对岸，也是非常气派。

![休整平整的绿化](./screennails/20250512_110400.webp)

岛屿很大，海南岛各特色亚热带风格树木有规则规划地栽种于海花岛之上，经前后搭配，既有雨林既视感，又不乏细致设计感，其间圆弧形水池更是增加了生活气息。

![酒店大堂](./screennails/20250511_132211.webp)

登岛后先是办理入住，整个过程依旧是非常快速。酒店也很有特色，大堂中木制雕刻及大量宽广沙发，在雨林玻璃幕墙内，显得很舒适大气，顶部细支蜂巢般连结星状灯光更是营造了一条弯曲银河效果。

![沙滩](./screennails/20250511_153808.webp)

既然是海岛，沙滩自然必不可少！白色细腻的大面积沙滩之上，一排沙滩椅在太阳伞之下，另有许多沙滩娱乐设施，诸如快艇、全地形车、足球等，可以想象一直到看到日落的整个下午的悠闲宁静之欢乐。

![岛另一边的仿古建筑物群](./screennails/20250511_150831.webp)

由于主岛形似六瓣花朵，每片花瓣沿岸都可以看到大海及临近花瓣沿岸景色，而每个花瓣又是不同特色，因此处处都是不一样的风景。瞧，对面是古朴中式岸边建筑物群。

![古风与现代风](./screennails/20250511_165135.webp)

也因此，还可以看到各色建筑物混搭的情况，中式古风、西洋古风、现代风等，常能同框出现，但似乎也不违和，倒是也有一番穿越效果，似乎大同包容之道亦在其中。

![中式建筑物](./screennails/20250511_164752.webp)

这些建筑当然也并不是一些无用的装饰，期内是各类餐饮购物等商铺，大多都是平日城市常见连锁品牌，因此在岛上也不用担心与世隔绝，大规模商业街似乎打造了一个新市镇，在景区观光之余也依然具有都市便利，甚至，岛上除了各类观光车外，还可以计时租用三轮或四轮游览电瓶车，非常方便。

![国际会议中心](./screennails/20250512_115436.webp)

白色飘逸与蓝色玻璃球相结合，12朵盛放的牡丹造型的国际会议中心设计得也是挺有趣的。

![船头长翅膀的雄狮](./screennails/20250511_172309.webp)

岛内另有一个大型城堡风的酒店，为了方便参观，虽然这次并无打算住在其中，但依然也有订其中房间。门口水池中一座大型雕塑——狮身鹰翼于宝船头，在后面数排高达欧式建筑物前，神武气势非凡，后面4个水中圆亭亦可顺弯曲水草状道路进入。

![旋转楼梯](./screennails/20250511_173910.webp)

酒店大堂也非常宏大，到处都是带有欧洲古堡风格的设计：蓝色花纹柱子、旋转木马似内廷、欧式花岗岩外观旋转楼梯、石膏款人体雕塑，浮夸的装潢让人眼前一亮。

![海景](./screennails/20250511_175749.webp)

从房间内，由于角度刚好面向岛屿外侧，可以看到更为宽广的海景，一望无际的对面有可能是越南，可以看到大洋中航行的船只。

## 夜景

![灯光秀](./screennails/20250511_213446.webp)

夜色之下，灯光照亮了海岛的美。夜晚有两场灯光秀，在哥特式建筑风的婚礼状元进行，五光十色动感十足的缤纷灯光秀投影在正面前的教堂式建筑上随着 BGM 翻转波动，音画效果算是比较震撼的。

![日本风格](./screennails/20250511_214038.webp)

每场灯光秀时间不是很长，大约 8min，旁边是日式建筑群，再旁边是泰式等等。

![海边夜景](./screennails/20250511_214738.webp)

也可以沿着海边行走，入口出的孪生主塔及其裙楼在灯光的勾勒下呈一条条优雅流线，旁边沙滩更是灯火通明传来阵阵歌声。
